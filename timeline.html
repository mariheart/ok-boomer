A Brief Timeline of the Epidemic

  • 1944 - first instance of Boomer generation
  • 1980 - first instance of Millenial generation
  • 1995 - first instance of Generation Z
  • September 3rd, 2015 - first instance of "Ok, boomer"
  • January 2019 - "Ok, boomer" enters mass media culture
  • Today - "Ok, boomer" now used as a retort against the older generation
